- 点击 Dashboard 右上角搜索图标

- 导出为 JSON 格式
- 导出为 Markdown 格式

#### 🎨 自定义主题色
位置：设置页面 - "主题颜色"区域
- 选择 8 种颜色之一
- 实时预览效果
