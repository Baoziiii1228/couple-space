### 使用方式
1. 点击"写日记"按钮
2. 在弹出的表单中，可以看到"快捷标签"区域
3. 点击任意标签，对应的文本会自动添加到内容输入框
4. 可以继续点击其他标签，内容会换行追加
5. 也可以手动编辑内容

### 技术实现
- 使用 Button 组件实现标签按钮
- 点击时通过 onClick 事件处理
- 自动判断是否需要换行
- 使用 `\n` 实现换行效果
