# 纪念日快速选择功能说明

## 功能概述

为纪念日功能添加了快速选择节假日的便捷功能，用户无需手动输入日期和选择emoji，只需点击预设按钮即可自动填充。

## 使用方式

### 在添加纪念日时

1. 点击"添加纪念日"按钮
2. 在表单中找到"快速选择节日"区域
3. 点击任意预设节日按钮
4. 系统自动填充：
   - **纪念日名称**：节日名称
   - **日期**：当前年份 + 节日日期
   - **Emoji**：对应的节日图标

### 在编辑纪念日时

编辑现有纪念日时，同样可以使用快速选择功能快速更改为预设节日。

## 技术实现

### 数据结构

```typescript
const presetDates = [
  { title: "元旦", date: "01-01", emoji: "🎉" },
  { title: "情人节", date: "02-14", emoji: "💕" },
  // ... 其他节日
];
```

### 自动填充逻辑

点击预设按钮时，系统会：
1. 获取当前年份（`new Date().getFullYear()`）
2. 组合成完整日期格式：`YYYY-MM-DD`
3. 同时填充标题和对应的emoji

```typescript
onClick={() => {
  const currentYear = new Date().getFullYear();
  setFormData({ 
    ...formData, 
    title: preset.title,
    date: `${currentYear}-${preset.date}`,
    emoji: preset.emoji
  });
}}
```

## UI设计

### 布局

快速选择区域采用 **3列网格布局**，每个按钮包含：
- Emoji图标
- 节日名称

- 使用 `glass` 效果保持与整体设计风格一致

## 用户体验优化

### 便捷性提升

**之前**：用户需要手动输入日期、选择emoji、输入标题
**现在**：一键填充所有信息，大幅减少操作步骤

### 适用场景

1. **快速添加常见节日**：情人节、圣诞节等
2. **批量创建纪念日**：一次性添加多个节日
3. **避免输入错误**：预设日期确保准确性

## 未来扩展
